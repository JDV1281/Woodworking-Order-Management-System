Woodworking Order Management System: Code Folder
Creator: Justin Verrick

This folder contains the main code files for the Woodworking Order Management System. The system was built using HTML, CSS, and JavaScript. It runs in a web browser and uses browser localStorage to save order information.

Files included:

- index.html: This file contains the main structure of the web page. It includes the page heading, order entry form, order summary section, and saved orders table.
- style.css: This file controls the design of the page. It handles the colors, spacing, layout, form styling, buttons, and table appearance.
- script.js: This file contains the JavaScript for the system. It handles adding new orders, checking required fields, calculating balance due, saving orders to localStorage, displaying saved orders, updating order status, deleting orders, and updating the order summary.

How to run the system:

Step 1. Open the 1_Code folder.
Step 2. Double-click the index.html file.
Step 3. The system will open in a web browser.
Step 4. Enter a sample woodworking order into the form.
Step 5. Click Save Order.
Step 6. The saved order will appear in the Saved Orders table.
Step 7. The order will stay saved after refreshing the page because it uses browser localStorage.

Notes:

- This project does not have and use a separate database. The order information is saved locally in the browser. 

**IMPORTANT** (If the browser data/history is cleared, the saved orders will be removed.) **IMPORTANT**
