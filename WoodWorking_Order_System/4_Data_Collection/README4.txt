Woodworking Order Management System: Data Collection Folder

This folder contains data for the Woodworking Order Management System. 
The data is used to show what kind of information the system will store when customer orders are entered.

The system collects information: 

- customer name 
- phone number
- email
- project type
- project description
- estimated cost
- amount paid
- balance due
- due date
- order status
- date created

Files included:

- sample_data.json: This file contains example woodworking order records that can be used for testing and demonstration.

Notes:

- The project does'nt use a separate database. 
- The system stores order records in browser localStorage.
- The sample data file is to show the structure of the data used by the system.
